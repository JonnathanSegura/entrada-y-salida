Entrada y Salida en Java

Este repositorio contiene programas básicos desarrollados en Java para practicar el manejo de entrada y salida estándar, impresión en consola, lectura de argumentos desde la línea de comandos y generación de patrones visuales.


Requisitos

Java 17

IntelliJ IDEA (o cualquier IDE para Java)

Git (opcional, para clonar el repositorio)

Ejecución

Clonar el repositorio

Abrir la carpeta en IntelliJ

Abrir el archivo .java que deseas ejecutar

Ejecutar el método main desde el botón Run del IDE

También puedes ejecutar los programas desde la terminal, por ejemplo:

javac Checkerboard.java
java Checkerboard

Archivos principales

Checkerboard.java → Imprime un tablero tipo ajedrez usando caracteres.

ShannonEntropy.java → Calcula la entropía de Shannon a partir de una secuencia de valores.

WorldMap.java → Visualiza un mapa basado en coordenadas de entrada estándar.
